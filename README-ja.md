## 🚀 概要

**AI BasePress** は、WordPress 開発のための **極限までシンプルなスターターテーマ** です。

特に以下の人のために設計されています：

* テーマ構造を **完全に把握したい開発者**
* ChatGPT などの **AI にカスタマイズを手伝わせたい人**
* クライアント案件や個人開発のために、**軽量・透明なベーステーマ** が欲しい人

既存のスターターテーマ（Underscores、Sage、Blockbase 等）とは違い、AI BasePress は：

* **最小構成（必要最低限）**
* **フラットなディレクトリ（複雑な階層なし）**
* **AI が読みやすい構造**
* **依存なし・ビルドツールなし**

という特徴を持つ、AI 開発時代向けのテーマです。

---

## 🎯 目的

1. **とにかくミニマルであること**
   WordPress の運用に本当に必要なテンプレートだけを保持。

2. **AI が100%理解できるテーマ構造**
   ファイルの内容・役割・命名を徹底統一。

3. **人間にとっても極めて扱いやすい**
   初心者〜中級者でもカスタマイズしやすい。

4. **外部依存ゼロ**
   CSSフレームワーク・JS依存・ビルドステップなし。

5. **将来の拡張にも耐えるベース**
   プラグイン対応版やブロック版への派生も可能。

## 📦 構成ファイル

AI BasePress に含まれるのは、以下の**最小限のテーマ構造**です：

```
ai-basepress/
├─ style.css          ← テーマ情報 + 最小リセット
├─ functions.php      ← 必要最低限のフックのみ
├─ index.php          ← フォールバック
├─ header.php
├─ footer.php
├─ sidebar.php
├─ single.php         ← 投稿ページ
├─ page.php           ← 固定ページ
├─ archive.php        ← ブログ/カテゴリ一覧
├─ 404.php
└─ /assets
     ├─ css/          ← 空（ここに独自CSSを配置）
     └─ js/           ← 空（ここに独自JSを配置）
```

**ブロックテーマではありません。**
**クラシックテーマの最小構成です。**

## 🧠 「AIフレンドリー」とは？

AI BasePress の最大の特徴は「AI によるカスタマイズを前提にした構造」であることです。

AI がテーマを扱うとき、以下の点が重要になります：

* **ファイル単位が小さいこと**
* **役割が明確であること**
* **命名が一貫していること**
* **ディレクトリ階層がフラットであること**
* **テーマ固有の複雑なロジックがないこと**

AI BasePress はこれらをすべて満たしているため、次のような指示が非常に成功しやすくなります：

> 「ポートフォリオ用のカスタム投稿タイプを追加して、
> archive-portfolio.php と single-portfolio.php を作って。」

> 「Figma のこのレイアウトに合わせてヘッダーを作り直して。」

AI が100%構造を把握しやすいため、**ミスが少なく、開発効率が圧倒的に高くなる**のが特徴です。

## 🛠️ 使い方

### 1. ダウンロード

このリポジトリをクローン or ZIP ダウンロードし、
`/wp-content/themes/` に配置します。

### 2. WordPress で有効化

管理画面 → 外観 → テーマ → AI BasePress を有効化

### 3. 自由にカスタマイズ

* header/footer の編集
* ページテンプレート追加
* スタイル・スクリプト追加
* デザインの組み込み
* AI にコード生成を依頼して追加する
  など、どんな用途にも使用できます。

AI BasePress は「壊れにくく、読みやすく、AI が勝手に壊すことも少ない構造」です。

## 🔌 プラグイン対応（方針）

AI BasePress の **コア版はプラグイン非対応（特殊処理なし）** です。

理由：

* プラグインに合わせると肥大化する
* 利用者ごとに必要なプラグインが異なる
* スターターテーマは軽さが正義

ただし今後、以下のような派生版を出す可能性があります：

| エディション                                 | 内容            |
| -------------------------------------- | ------------- |
| **AI BasePress – Plugin-Ready**        | 主要プラグイン対応版    |
| **AI BasePress – WooCommerce Edition** | WooCommerce対応 |
| **AI BasePress – Block Theme Edition** | ブロックテーマ対応     |

需要に応じて展開します。
